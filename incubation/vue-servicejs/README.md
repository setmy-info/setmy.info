## vue-servicejs

VueJS ServiceJS plugin

### Why?

To add service injection into components.

### Setup

npm install

or

bower install

### Run server for examples

npm run start

or

npm start

or

mvn jetty:run

http://localhost:3000/

### Unit testing

npm run unit

### Run example program
